Binaries

ActivePerl/Win32

PPM binary packages are available for ActivePerl 6xx, ActivePerl 8xx, ActivePerl 6xx with Unicode (for NT/2000/XP) ActivePerl 8xx with Unicode (for NT/2000/XP).
Please read the included README.txt for installation instructions.
The corresponding developement packages for ActivePerl 6xx, ActivePerl 8xx, ActivePerl 6xx with Unicode, ActivePerl 8xx with Unicode should only be downloaded by people interested in developing/compiling (and not just using) wxPerl extensions using XS, such as Wx::ActiveX.

Red Hat Linux

Binary RPMs for wxWidgets 2.4.2, Red Hat Linux 8.0 are available.

Mac OS X 10.3 (Panther)

A DMG image for Mac OS X 10.3 (Panther) is available. Please note that unlike all other binaries it is compiled against wxWidgets 2.5.1.
