Binaries

ActivePerl

PPM binary packages for Windows, Linux and Mac OS X can be downloaded from the wxPerl PPM Site

Linux

wxPerl is available as a native package in Debian, Fedora, Gentoo, Mandriva, Ubuntu and other major Linux distributions.

Documentation

Install the Wx::Demo package from CPAN.

Documentation fo wxPerl is available as TeX, HTML, and MS HTML Help

  • Microsoft HTML Help (CHM) format
  • ZIP file containing HTML documentation
  • TeX sources used to build the documentation

Source

Source code for wxPerl is available on CPAN.

Subversion

The bleeding-edge sources are available through Subversion.

Anonymous Subversion access

svn co http://svn.code.sf.net/p/wxperl/code/wxPerl/trunk wxPerl-trunk

Browse Subversion sources

SourceForge offers a Subversion browser interface that can be used to look at Subversion sources, compare the differences between revisions and get the most up to date revision of a single file.
