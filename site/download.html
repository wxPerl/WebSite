Binaries

ActivePerl/Win32

PPM binary packages are available for
  • ActivePerl 8xx
  • ActivePerl 8xx with Unicode (for NT/2000/XP)
Please read the included README.txt for installation instructions.
The corresponding developement packages for
  • ActivePerl 8xx
  • ActivePerl 8xx with Unicode
should only be downloaded by people interested in developing/compiling (and not just using) wxPerl extensions using XS, such as Wx::ActiveX.

Red Hat Linux

  • wxWidgets 2.6.2, Red Hat Linux 8.0, GTK 2

Source

Source code for wxPerl is available in various formats:
  • Standard CPAN package (also available on CPAN)
  • RPM source package

CVS

The bleeding-edge sources are available through CVS.

Anonymous CVS access

cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/wxperl login
password:<press enter>
cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/wxperl co wxPerl

Browse CVS sources

SourceForge offers a CVS browser interface that can be used to look at CVS sources, compare the differences between revisions and get the most up to date revision of a single file.

Anonymous CVS snapshots

Daily CVS snapshots are also available in .tar.gz, .tar.bz2 and .zip format.
