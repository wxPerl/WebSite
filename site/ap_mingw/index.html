ActivePerl + MinGW

This page documents a procedure to build Perl modules for ActivePerl using MinGW ( gcc for Win32 ).

Last update: May 8 2001

Download (size 10k)

This zip contains ExtUtils::FakeConfig 0.02, the installation instructions, and a small script to automate import library creation.

Requirements

  • MinGW GCC 2.95.2-1
  • ActivePerl ( tested builds 616, 620; might work with earlier 6xx builds)
  • dmake
