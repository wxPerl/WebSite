API differences (WARNING: this list is incomplete)

all classes

Everywere you can pass a Wx::Point or Wx::Size object as a parameter, you can pass a reference to a 2 element array ( like [2, 3] ): this saves some typing (and is also a bit faster).

There is no two step construction.

You may pass a scalar as client data.

wxChoice

new
takes an array reference instead of 'n' and 'choices'

wxComboBox

new
takes an array reference instead of 'n' and 'choices'

wxDC

DrawSpline
takes an array of Wx::Point objects
GetClippingBox
takes no parameters and returna a 4 element array
GetPixel
takes only 'x' and 'y' parameters, returns a Wx::Colour object
GetSize
returns a 2 element array
GetTextExtent
takes only the 'string' and (optionally) 'font' parameters always returns a 4 element array
GetUserScale
returns a 2 element arrya

wxImageList

GetSize
takes an 'index' parameter and returns a 3 element array

wxLayoutConstraints

use it like this: $constraint->centreX->Absolute( 100 ); (much like as in C++).

wxListBox and wxCheckListBox

Methods taking an array of strings and the number of elements in the array will take only an array reference.

wxLocale

Use it as in C; this module exports a '_' function that does the job of the '_' C macro. ( use Wx qw(_) )

wxMenu

FindItem
takes only a parameter if it is a string returns the corresponding id, otherwise returns a 2 element array

wxPoint

Has 2 accessor methods ( 'x' and 'y' ); $p->x and $p->y will retrieve the value; $p->x( $newval ) and $p->y( $newval ) will set the value.

wxRadioBox

Methods taking an array of strings and the number of elements in the array will take only an array reference.

wxRect

Has 4 accessor methods ( 'x', 'y', 'width', 'height' ); $p->x, $p->width, ... will retrieve the value; $p->x( $newval ), $p->height( $newval ), ... will set the value.

wxSize

Has 2 accessor methods ( 'width' and 'height' ); $p->width and $p->height will retrieve the value; $p->width( $newval ) and $p->height( $newval ) will set the value.

wxTextCtrl

GetSelection
has no parameters, returns a 2 element array (x,y)
PositionToXY
takes 1 parameter, returns a 2 element array

wxWindow

GetClientSize
GetClientSizeXY takes no arguments and returns a 2 element array (x,y), GetClientSize takes no arguments and returns a Wx::Size object
GetPosition
has no parameters, returns a 2 element array (x,y)
GetSize
GetSizeXY takes no arguments and returns a 2 element array (x,y), GetSize takes no arguments and returns a Wx::Size object
GetTextExtent
takes only the 'string' and (optionally) 'font' parameters always returns a 4 element array
